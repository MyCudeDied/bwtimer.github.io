# CSTimer Makeover

Opinionated chrome extension for re-styling [CSTimer](https://cstimer.net). You can install it [on the Chrome store](https://chrome.google.com/webstore/detail/cstimer-makeover/omchdkmkmadmhkaamhpojgldknpihgcg).
